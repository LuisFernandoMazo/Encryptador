# 🔐 Encriptador de Texto

![Encriptador de Texto](assets/img/detective.png)

## Descripción

Bienvenido al **Encriptador de Texto**, una aplicación web interactiva diseñada para encriptar y desencriptar texto utilizando reglas específicas. Ideal para aquellos que desean practicar la encriptación de texto de manera simple y disfrutar de una experiencia de usuario agradable y visualmente atractiva.

## 🚀 Características Principales

- 🔐 **Encriptación y Desencriptación**: Transformación del texto ingresado en un formato encriptado y la capacidad de revertir el proceso.
- 🖼️ **Interfaz de Usuario Intuitiva**: Diseño limpio con una estructura clara, que incluye un modal de bienvenida, secciones de encabezado y pie de página, y un área principal para la interacción del usuario.
- 👋 **Modal de Bienvenida**: Modal inicial que da la bienvenida al usuario con una animación de carga progresiva.
- ✅ **Validación de Texto**: Asegura que el texto ingresado solo contenga letras minúsculas y sin acentos, notificando al usuario en caso de errores.
- 📋 **Función de Copiar**: Permite al usuario copiar el texto encriptado o desencriptado al portapapeles con un solo clic.
- 📱 **Compatibilidad con Dispositivos**: Adaptado para funcionar correctamente en diferentes tamaños de pantalla y dispositivos.

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Para la estructura del contenido.
- **CSS3**: Para el diseño y la disposición de los elementos visuales.
- **JavaScript**: Para la funcionalidad interactiva y la lógica de encriptación/desencriptación.

🌟 Uso
1. Ingresar Texto: Ingresa el texto en el área de entrada proporcionada.
2. Encriptar/Desencriptar: Utiliza los botones correspondientes para encriptar o desencriptar el texto.
3. Copiar Texto: Copia el resultado encriptado o desencriptado al portapapeles con el botón de copiar.
